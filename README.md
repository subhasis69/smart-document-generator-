🚀 Smart Document Generator

A modern web-based application to create, preview, and convert documents into PDF instantly. Supports Certificates, Invoices, Resumes, and custom HTML files with a clean UI and live preview.

✨ Features 📄 Generate Certificates dynamically 🧾 Create Invoices with items, GST & totals 👤 Build simple Resumes 💻 Convert HTML → PDF 👁️ Real-time Live Preview 💾 Save & manage invoices (Local Storage) 🌙 Dark / Light mode toggle ⚡ Smooth UI with sidebar navigation 🛠️ Tech Stack Layer Technology Frontend HTML, CSS, JavaScript Backend Node.js, Express Features Fetch API, FileReader, LocalStorage 📂 Project Structure Smart-Document-Generator/ │ ├── index.html ├── server.js ├── package.json ⚙️ Installation & Setup 1️⃣ Clone the repository git clone https://github.com/your-username/smart-document-generator.git cd smart-document-generator 2️⃣ Install dependencies npm install 3️⃣ Run the server node server.js 4️⃣ Open in browser http://localhost:3000 📌 How to Use Select document type from sidebar Fill required details Click Generate View live preview Convert to PDF and download 🔥 Key Functionalities ✅ Certificate Generator Auto formatted certificate Dynamic data insertion ✅ Invoice Generator Add multiple items GST calculation Save & load invoices ✅ Resume Builder Simple structured resume layout ✅ HTML to PDF Upload or paste HTML Live preview Convert to downloadable PDF 💡 Future Improvements 📦 Multiple templates ☁️ Cloud storage 📄 Export to DOCX 🎨 Theme customization 📱 Mobile responsive design 🔐 User authentication
